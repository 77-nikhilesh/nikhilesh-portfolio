## 📌 About This Project

This portfolio was **initially designed and developed from scratch** using core web technologies.  
The goal was to create a **clean, professional, and high-performance portfolio** that effectively presents my work to recruiters.

After building the core structure, I leveraged **AI tools to enhance and optimize the project** by:

- Improving UI/UX consistency and layout
- Refining content for better clarity and impact
- Optimizing performance and reducing unnecessary code
- Enhancing responsiveness and interactivity
- Improving overall code structure and maintainability

> ⚡ This project reflects both my development skills and my ability to use modern tools like AI effectively to improve real-world applications.

## 🛠️ Tech Stack

- **HTML5** – Structure and semantic layout  
- **CSS3** – Modern UI design, animations, responsiveness

- **Git & GitHub** – Version control and project management
